Optional shape for [notify](./notify) — a thinner convenience API over
[toast](./toast). `kind` maps onto PrimeVue toast severities; `message` maps
to `summary`; `detail` is optional supplementary text.

| <a id="detail"></a> `detail?` | `string` | Optional supplementary detail line. |

| <a id="life"></a> `life?` | `number` | Auto-dismiss delay in ms. Defaults to PrimeVue's `life`. |
