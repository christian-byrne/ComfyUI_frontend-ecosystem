Defined in: [src/extension-api/node.ts:43](https://github.com/Comfy-Org/ComfyUI_frontend/blob/59a2e1ce81b1ae31a7991a24d5b4ecb0a88d27ad/src/extension-api/node.ts#L43)

A 2D size as `[width, height]`.

**Immutable tuple per D-immutability-enforcement (Hybrid C).** Attempts to
mutate via `node.getSize()[0] = X` raise a TypeScript error. Use
[NodeHandle.setSize](../interfaces/NodeHandle.md#setsize) to resize the node.
