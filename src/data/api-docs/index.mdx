# @comfyorg/comfyui-frontend

@comfyorg/extension-api — Public Extension API for ComfyUI

This barrel is the published package entry point. Every export here is
part of the public contract that extension authors depend on.

Import directly — no dependency on `window.app` at module evaluation time:

```ts
import { defineNode, defineExtension } from '@comfyorg/extension-api'
```

## API surface overview

| Export | Purpose |
|--------|---------|
| `defineNode` | Register a node-scoped extension (the primary entry point) |
| `defineExtension` | Register an app-scoped extension (init, setup, shell UI) |
| `onNodeMounted`, `onNodeRemoved` | Implicit-context lifecycle hooks (call inside nodeCreated) |
| `NodeHandle` | Controlled access to node state and events |
| `WidgetHandle` | Controlled access to widget state and events |
| `WidgetBeforeQueueEvent` | Pre-queue validation event — call `reject(msg)` to cancel |
| `SlotInfo` | Read-only slot snapshot |
| Shell UI types | `SidebarTabExtension`, `BottomPanelExtension`, `CommandManager`, etc. |
| Identity helpers | `NodeLocatorId`, `NodeExecutionId`, parsers, type guards |

## Identity (D20)

Handles expose `id: string` and `equals(other)` — the 90% case never needs
a branded type. The runtime `*EntityId` brands (`NodeEntityId`,
`WidgetEntityId`, `SlotEntityId`) are an internal storage concern and are
NOT re-exported from this barrel. Protocol-boundary identifiers
(`NodeLocatorId` from workflow JSON, `NodeExecutionId` from websocket
frames) remain public because authors **receive** them from event
payloads.

## API style

The public API is **event + getter/setter**, not signals. Vue reactivity is
the internal engine; extension authors never import from Vue or use
`ref`/`computed`/`effect` directly. State is read via methods (`getValue()`,
`getPosition()`), mutated via command-dispatch methods (`setValue()`,
`setPosition()`), and observed via typed event subscriptions (`on('executed', fn)`).
Read-only invariants (set at construction, never change) are exposed as
accessors (`get entityId`, `get type`).

## Barrel-file rule exception

ComfyUI_frontend AGENTS.md rule #19 normally forbids barrel files in `/src`.
This barrel is the **published package entry point** — not an internal
re-export — and is the explicit exception documented in AGENTS.md.

## Interfaces

| Interface | Description |
| ------ | ------ |
| [AboutBadgeExtension](interfaces/AboutBadgeExtension.md) | Public arg type for [defineAboutBadge](functions/defineAboutBadge.md). A badge that appears on the About page (linked label + icon + optional severity). Mirrors v1 `extension.aboutPageBadges[]`. |
| [DisposableHandle](interfaces/DisposableHandle.md) | Handle returned by every `defineX`. Call `dispose()` to remove the registration. Idempotent and synchronous — see module-level "Dispose contract" notes. |
| [ExecutionEventPayloads](interfaces/ExecutionEventPayloads.md) | - |
| [ExtensionOptions](interfaces/ExtensionOptions.md) | Options for the global `defineExtension` entry point. Covers extension-wide lifecycle and shell UI contributions. |
| [GraphEventPayloads](interfaces/GraphEventPayloads.md) | Per-namespace event payload map. **Augment via TS module augmentation** to narrow payloads for canonical events. Until D5 ships, all payloads default to `unknown`. |
| [HotkeyExtension](interfaces/HotkeyExtension.md) | Public arg type for [defineHotkey](functions/defineHotkey.md). A hotkey binds a key combination (already-registered command) to fire on press. Mirrors v1 `extension.keybindings[]` which used the internal `Keybinding` shape. |
| [~~NodeBeforeSerializeEvent~~](interfaces/NodeBeforeSerializeEvent.md) | Payload for `node.on('beforeSerialize', handler)`. |
| [NodeExecutedEvent](interfaces/NodeExecutedEvent.md) | Payload for `node.on('executed', handler)`. |
| [NodeExtensionOptions](interfaces/NodeExtensionOptions.md) | Options for `defineNode`. Describes an extension that reacts to node lifecycle events. |
| [NodeHandle](interfaces/NodeHandle.md) | Controlled surface for node access. Reads query the ECS World; writes dispatch commands. Events are Vue-reactive watches on World components. |
| [NotifyOptions](interfaces/NotifyOptions.md) | Optional shape for [notify](functions/notify.md) — a thinner convenience API over [toast](variables/toast.md). `kind` maps onto PrimeVue toast severities; `message` maps to `summary`; `detail` is optional supplementary text. |
| [ServerEventPayloads](interfaces/ServerEventPayloads.md) | - |
| [ToastMessageOptions](interfaces/ToastMessageOptions.md) | Defines message options in Toast component. |
| [ToolbarButtonExtension](interfaces/ToolbarButtonExtension.md) | Public arg type for [defineToolbarButton](functions/defineToolbarButton.md). A button that appears in the action bar (top of the canvas area). Wraps the v1 `extension.actionBarButtons[]` shape with an `id` so each registration is independently disposable. |
| [WidgetBeforeQueueEvent](interfaces/WidgetBeforeQueueEvent.md) | Payload for `widget.on('beforeQueue', handler)`. |
| [WidgetBeforeSerializeEvent](interfaces/WidgetBeforeSerializeEvent.md) | Payload for `widget.on('beforeSerialize', handler)`. |
| [WidgetExtensionOptions](interfaces/WidgetExtensionOptions.md) | Options for `defineWidget`. Registers a custom widget type that renders through the mount-lifecycle seam (Axiom A12 / D-widget-converge). |
| [WidgetHandle](interfaces/WidgetHandle.md) | Controlled surface for widget access. Backed by ECS `WidgetValue` and `WidgetIdentity` components in the World. Reads query components directly; writes dispatch commands (undo-able, serializable, validatable). |
| [WidgetMountContext](interfaces/WidgetMountContext.md) | Context passed to a widget's `mount()` function. |
| [WidgetOptionChangeEvent](interfaces/WidgetOptionChangeEvent.md) | Payload for `widget.on('optionChange', handler)`. |
| [WidgetOptions](interfaces/WidgetOptions.md) | Options passed to `node.addWidget()` when creating a new widget. |
| [WidgetPropertyChangeEvent](interfaces/WidgetPropertyChangeEvent.md) | Payload for `widget.on('propertyChange', handler)`. |
| [WidgetValueChangeEvent](interfaces/WidgetValueChangeEvent.md) | Payload for `widget.on('valueChange', handler)`. |
| [WorkbenchEventPayloads](interfaces/WorkbenchEventPayloads.md) | - |

## Type Aliases

| Type Alias | Description |
| ------ | ------ |
| [AsyncHandler](type-aliases/AsyncHandler.md) | A typed async-capable event handler. Only valid for events that explicitly support async handling (currently only `beforeSerialize`). |
| [BottomPanelExtension](type-aliases/BottomPanelExtension.md) | - |
| [CommandDefinition](type-aliases/CommandDefinition.md) | Public arg type for [defineCommand](functions/defineCommand.md). Alias of the v1 `ComfyCommand` shape exported from `@/stores/commandStore`, re-surfaced under a public name so authors do not import from the internal `stores/` path. |
| [Handler](type-aliases/Handler.md) | A typed event handler function. |
| [NodeExecutionId](type-aliases/NodeExecutionId.md) | An execution identifier representing a node's position in nested subgraphs. Also known as ExecutionId in some contexts. |
| [NodeLocatorId](type-aliases/NodeLocatorId.md) | A globally unique identifier for nodes that maintains consistency across multiple instances of the same subgraph. |
| [Point](type-aliases/Point.md) | A 2D point as `[x, y]`. |
| [SettingDefinition](type-aliases/SettingDefinition.md) | Public arg type for [defineSetting](functions/defineSetting.md). Re-surfaces the existing `SettingParams` shape (from `@/platform/settings/types`) under a public name so authors do not import from the internal `platform/` path. |
| [SidebarTabExtension](type-aliases/SidebarTabExtension.md) | - |
| [Size](type-aliases/Size.md) | A 2D size as `[width, height]`. |
| [SlotInfo](type-aliases/SlotInfo.md) | - |
| [ToastManager](type-aliases/ToastManager.md) | - |
| [Unsubscribe](type-aliases/Unsubscribe.md) | Cleanup function returned by `on()` — call to remove the listener. |
| [WidgetCleanup](type-aliases/WidgetCleanup.md) | Cleanup function returned from a widget's `mount()`. Fires exactly once, when the widget entity is destroyed. **Does NOT fire on host remount** (graph↔app mode, subgraph promotion, `<KeepAlive>` shuffle) — use [WidgetMountContext.onBeforeRemount](interfaces/WidgetMountContext.md#onbeforeremount) / [WidgetMountContext.onAfterRemount](interfaces/WidgetMountContext.md#onafterremount) for those. |
| [WidgetMountFn](type-aliases/WidgetMountFn.md) | Mount function for a widget. Called once when the widget is first attached to a node host in the DOM. Returns an optional cleanup function that fires on widget destruction. |
| [WidgetValue](type-aliases/WidgetValue.md) | The union of all legal widget scalar values. Complex widgets (DOM, canvas) may return their own serializable shapes. |
