Register a hotkey binding. Returns a [DisposableHandle](./disposablehandle) — call
`handle.dispose()` to unbind.

The hotkey targets a command id; register the command separately via
[defineCommand](./definecommand). The `keys` string is parsed via the standard
key-combo grammar (`mod+k`, `ctrl+shift+f`, etc.).
